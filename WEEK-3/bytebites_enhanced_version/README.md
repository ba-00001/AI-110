# ByteBites Enhanced Version

This folder contains the enhanced ByteBites version. It keeps the original Python class architecture, then adds a browser UI, an AI assistant panel, local API routes, and a Vercel-ready structure for future deployment.

## Included Files

- all core design files from the original assignment
- `models.py`
- `test_bytebites.py`
- `index.html`, `styles.css`, `script.js`
- `bytebites_backend.py`
- `api/menu.py`
- `api/assistant.py`
- `ai_server.py`
- `.env.example`

## What This Version Adds

- a browser-based menu and order interface
- more menu items for a stronger demo
- search, sorting, filtering, and order history
- an AI assistant panel
- local fallback recommendations if Gemini is unavailable
- backend structure that is easier to adapt for Vercel
