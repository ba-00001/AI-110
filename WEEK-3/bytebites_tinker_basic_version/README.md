# ByteBites Basic Version

This folder contains the basic Week 3 assignment version of ByteBites. It stays close to the original requirements by focusing on the four main Python classes and the required design files, without the browser app or AI additions.

## Included Files

- `bytebites_spec.md`
- `draft_from_copilot.md`
- `bytebites_design.md`
- `.github/agens/bytebites_design_agent.md`
- `models.py`
- `test_bytebites.py`
- `pytest.ini`

## What This Version Includes

- `Customer`, `MenuItem`, `Menu`, and `Order`
- category filtering
- popularity sorting
- order total calculation
- a small manual demo inside `models.py`
- pytest coverage for the main required behaviors
